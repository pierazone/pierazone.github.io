PIERA-ZONE makes it easier for educators and institutions to conduct fair digital exams for FREE with an easy to use interface and clear workflow of the entire process from creation, conduction, evaluation to sharing and working with results. It has been created by keeping individual use in mind, but we do provide a basic setup for an institution with a basic hierarchy who would like to have a login and log of user activity.



KEY FEATURES



Candidates can not perform any other activity on the system than the exam, doing otherwise leads to immediate termination of the exam without prior warning, thus reducing a possibility of malpractice.
Examiners can create exams by giving in the basic details and the rest is automated. Sharing the exam with the candidates can be done by sharing a single file via any means.
Questions of type single correct answer and multiple correct answers (out of 4 options), numerical or text, and/or a combination of all is facilitated.


Exams can be conducted without the need to have internet connection as all the processes happen locally. An attempt to email the responses of candidates will be made to the email ID provided while creating the exam, but if the internet is disconnected or this feature has been disabled (or any other reason of failure of the same), candidates can find a local copy of their responses which can be shared later as instructed.
Evaluate the results with a tap of a button which automatically generates individual results in the form of PDF that can be shared by any means or use our built in feature to send it via email automatically. You can also modify the exam key and re-evaluate multiple times.
Candidates have a choice to mark questions for review which can later be visited and revised. Color indicators have been used to indicate if they have -
  • Visited but not attempted
  • Attempted
  • Attempted and marked for review
  • Didn’t attempted but marked for review
They can navigate to any question using the navigation pane consisting of option menus for the same.


All the results after final evaluation are stored in the database, which can be viewed through our application and can also be exported as an excel file.
Easily switch between usage type of individual or an organisation.
Exams and reading the instructions before an exam are time bound as specified during the exam creation, after which the exam will auto submit and launch the exam respectively.
